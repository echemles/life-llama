
=====================

### Tech Stack

* Node
* ReactJS
* Sass
* Webpack
* Express


### Running App

To run locally, install and run using the commands below:

```
npm install
brew install libpng
npm run start
```

Open up a browser tab and navigate to [localhost:3000](http://localhost:3000/).
